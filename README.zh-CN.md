# Vue + Vite 应用模板

基于 Vite 的现代 Vue 3 应用模板，支持 TypeScript 和快速热模块替换（HMR）。

> [English](./README.md) | 简体中文

## 🚀 特性

- ⚡️ **Vite** - 极速的开发体验和即时热更新
- 💚 **Vue 3** - Composition API 和 `<script setup>` 语法
- 🔷 **TypeScript** - 完整的类型安全
- 📦 **组件化** - 模块化组件架构
- 🎨 **CSS 支持** - 内置 CSS 和作用域样式
- 🚀 **优化构建** - 生产级构建，支持 tree-shaking

## 🏁 快速开始

### 安装

```bash
# 安装依赖
npm install
```

### 开发

```bash
# 启动开发服务器 http://localhost:3000
npm run dev
```

应用将自动在默认浏览器中打开。

### 构建

```bash
# 构建生产版本
npm run build

```
